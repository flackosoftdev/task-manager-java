# Task Manager Java

A simple Java practice project for learning core backend fundamentals.

## Topics covered
- OOP
- ArrayList
- HashMap
- Exceptions
- Inheritance
- Interface
- Static methods
- Enum
- Abstract classes

## Project structure
- `src/oop` — Task and TaskManager
- `src/mappractice` — HashMap practice
- `src/exceptionspractice` — exception handling practice
- `src/inheritancepractice` — inheritance practice
- `src/interfacepractice` — interface practice
- `src/staticpractice` — static methods practice
- `src/enumpractice` — enum practice
- `src/abstractpractice` — abstract class practice
